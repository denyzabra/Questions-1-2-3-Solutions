# Coding Challenge Solutions - README

These are my solutions to three coding challenges, all written in PHP. Below are brief descriptions of each challenge and instructions on how to access and see the output.

## Challenge 1: Fibonacci Numbers

### Description:
Sort an array and return an array of Fibonacci numbers starting from the seventh number.

### Solution:
- The PHP code for this challenge is located in `sort_and_fibonacci.php`.

### How to Run:
1. Make sure you have PHP and XAMPP installed.
2. Start your XAMPP server.
3. Open a web browser and navigate to `http://localhost/project_directory/sort_and_fibonacci.php`.
4. You will see the sorted array and the seventh Fibonacci number and beyond displayed in the browser.

## Challenge 2: Extract Information from Text

### Description:
Extract an email address, phone number, and URL from a given paragraph using regular expressions and without regular expressions.

### Solution:
- The PHP code for this challenge is located in `extract_data.php`.

### How to Run:
1. Make sure you have PHP and XAMPP installed.
2. Start your XAMPP server.
3. Open a web browser and navigate to `http://localhost/project_directory/extract_data.php`.
4. You will see the extracted email address, phone number, and URL both with and without regular expressions displayed in the browser.

## Challenge 3: Text File Processing

### Description:
Read text from a file and perform tasks like removing duplicates and extracting punctuation marks.

### Solution:
- The PHP code for this challenge is located in `file_processing.php`.

### How to Run:
1. Make sure you have PHP and XAMPP installed.
2. Start your XAMPP server.
3. Open a web browser and navigate to `http://localhost/project_directory/file_processing.php`.
4. You will see the results of the tasks performed on the text file displayed in the browser.
